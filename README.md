# QtLearning

本项目是阅读豆子(devbean)老师的 Qt 学习之路2, 时根据博客内容练习以及实现的代码.

Qt 使用的版本为 4.8.7, 编译器使用为 MinGW 4.8.2

QJSON 章节后转用版本 5.12.12，编译器使用版本为 MinGW 7.3.0
